# Apache Beam Data Engineering Exercise

This demonstrates core Apache Beam concepts implemented in Google Colab.

### Implemented Features
- Map and Filter Transforms  
- ParDo (Custom DoFn)  
- Composite Transform (Reusable PTransform)  
- Partitioning  
- Windowing (Fixed Windows)  
- Pipeline I/O (CSV Read & Write)  
- ML Inference using Beam (Extra Credit)
